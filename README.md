<h2>Usage</h2>
<h3>Prerequisites</h3>

<h3>MongoDB</h3>
<h4>Node ^20.0.0</h4>
<h4>npm</h4>

Server-side usage (PORT: 5555)

  <p>cd backend   // go to server folder </p>
  <p> npm install   </p>
  <p> npm start</p>

Client-side usage:
<p>cd frontend</p>
<p>npm run dev</p>
