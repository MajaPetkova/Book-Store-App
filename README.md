<h2>Usage</h2>
<h3>Prerequisites</h3>

<h3>MongoDB</h3>
<h3>Node ^20.0.0</h3>
<h3>npm</h3>

Server-side usage (PORT: 5555)
   cd backend   // go to server folder
   npm install
   npm start

Client-side usage:
   cd frontend
   npm run dev
